README:

The loader can be flagged by anti virus because it has to get access to the game files to draw ESP and use aimbot etc.

1. disable anti-virus
2. download the Epitome loader zip file
3. unzip it , password: 7777
4. launch the "setup"
5. wait for loader to show "Injection Complete" message
6. start game
7. enjoy


List of features contains:
ESP - wallhack, skeleton, cham
Aimbot - high configurability, where to aim and smoothing
quick draw - pulls out weapons faster
quick scope - scopes in faster
nearby enemy alert
enemy player action - shows whether they are scoping in, reloading, crouching etc.
